"Настройка TFTP для разработки на NanoPi M4 (RK3399)
==================================================

1. Созданные файлы:
   - deploy_kernel_tftp.ps1 - Основной скрипт развертывания ядра через TFTP
   - start_tftp_server.ps1 - Скрипт запуска TFTP сервера
   - setup_tftp_environment.ps1 - Скрипт полной настройки TFTP окружения
   - Tftpd64.ini - Конфигурационный файл для Tftpd64
   - TFTP_SETUP_INSTRUCTIONS.md - Подробные инструкции по настройке
   - README_TFTP_SETUP.md - Документация по использованию

2. Настройка TFTP сервера:
   - Директория TFTP: C:\tftpboot
   - IP-адрес сервера: 192.168.1.166
   - Порт: 69 (стандартный TFTP порт)
   - Файл ядра: kernel_neuro.bin

3. Настройка клиента (NanoPi M4 / U-Boot):
   В консоли U-Boot выполните следующие команды один раз:
   
   setenv ipaddr 192.168.1.10
   setenv serverip 192.168.1.166
   setenv bootfile kernel_neuro.bin
   setenv netboot 'tftp ${loadaddr} ${bootfile}; go ${loadaddr}'
   saveenv

4. Использование:
   После настройки вы можете загружать ядро командой:
   run netboot

5. Автоматизация:
   Для автоматического развертывания используйте скрипт:
   .\deploy_kernel_tftp.ps1

   Этот скрипт выполняет следующие действия:
   - Компилирует ядро (если есть Makefile.neuro)
   - Копирует бинарный файл в директорию TFTP
   - Проверяет состояние TFTP сервера
   - Отправляет команду перезагрузки через UART (опционально)

6. Преимущества TFTP:
   - Время загрузки ядра: 2 секунды вместо 3-5 минут
   - Нет необходимости выключать плату и извлекать SD-карту
   - Автоматизация процесса развертывания
   - Упрощение тестирования изменений в коде

7. Устранение неполадок:
   Если возникают проблемы:
   1. Проверьте, что TFTP сервер запущен и слушает порт 69
   2. Убедитесь, что файл kernel_neuro.bin находится в директории C:\tftpboot
   3. Проверьте сетевое подключение между ПК и NanoPi
   4. Убедитесь, что настройки IP-адресов в U-Boot корректны

8. Дополнительные возможности:
   Скрипт deploy_kernel_tftp.ps1 поддерживает параметры:
   -AutoReboot - автоматическая отправка команды загрузки через UART
   -ComPort - указание COM-порта для UART
   -BaudRate - указание скорости передачи данных через UART

   Пример использования с параметрами:
   .\deploy_kernel_tftp.ps1 -AutoReboot -ComPort "COM4" -BaudRate 115200
"
